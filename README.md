# Распознование лиц по изображению

Мини-система распознавания лиц, разработанная как учебный проект.
Система определяет человека по фотографии и выдаёт сообщение:

* «Hi, {name}!» — лицо известно
* «Error» — лицо неизвестно

---

## Функции

* Загрузка изображения (файл или веб-камера)
* Детектирование лица: Haar Cascades
* Нормализация: `128x128`, grayscale, выравнивание контраста
* Извлечение признаков: HOG (можно LBPH/PCA)
* Классификация: SVM (или k-NN)
* Вывод результатов: имя + вероятность, цвет рамки (зелёная/красная)
* Дополнительно: озвучивание, регистрация нового пользователя, журнал входов

---

## Использование

1. **Обучение модели**

```bash
python train.py
```

2. **Распознавание лиц**

```bash
python recognize.py
```

* Результаты сохраняются в `results/`
* Выводятся рамки и имя с вероятностью
